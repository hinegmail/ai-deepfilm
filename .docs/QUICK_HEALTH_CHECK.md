# 快速健康检查指南 (30 秒)

## 打开健康检查

1. 点击 **⚙️ 模型配置** 按钮
2. 点击 **"健康检查"** 标签页

## 检查所有提供商 (推荐)

点击右上角 **"全部检查"** 按钮，等待完成。

```
✅ 连接正常    = API 工作正常，可以继续使用
⚠️ API Key 问题 = 需要配置或更新 API Key
❌ 连接失败    = 检查 URL 或网络连接
⏱️ 请求超时    = 网络缓慢，稍后重试
```

## 常见问题速查

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| ⚠️ API Key 问题 | API Key 未配置/无效 | 返回"全局配置"重新配置 |
| ❌ 连接失败 | URL 错误或网络问题 | 检查"API 提供商"中的 URL |
| ⏱️ 请求超时 | 网络慢或 API 过载 | 稍后重试或检查网络 |

## 验证特定模型

1. 展开有问题的提供商 (点击 ▼)
2. 找到对应的模型
3. 点击 **"验证"** 按钮

## 添加新提供商后检查

1. 返回 "API 提供商" 标签页添加新提供商
2. 返回 "健康检查" 标签页
3. 为新提供商点击 ⚡ 进行检查

## 响应时间参考

- **< 500ms**: 很好 ✅
- **500-2000ms**: 正常 ✓
- **2-10s**: 缓慢 ⚠️
- **> 10s**: 超时 ❌
