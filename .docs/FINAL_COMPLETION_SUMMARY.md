## 🎯 用户使用流程

### 场景 1: 首次使用

```
1. 打开模型配置 ⚙️
2. 选择 "全局配置" 标签页
3. 输入 API Key
4. 返回 "健康检查" 标签页
5. 点击 "全部检查" 验证
6. 确保所有提供商都 ✅ 连接正常
7. 开始使用应用
```

### 场景 2: 添加 AGNES AI

```
1. 打开模型配置 ⚙️
2. 选择 "全局配置"
3. 配置 AGNES API Key
4. 返回 "健康检查"
5. 查看 AGNES AI 状态 (应该已显示)
6. 点击 ⚡ 验证连接
7. 在对话模型中选择 AGNES 模型
```

### 场景 3: 添加其他 OpenAI 兼容 API

```
1. 打开模型配置 ⚙️
2. 选择 "API 提供商" 标签页
3. 点击 "添加提供商"
4. 填写 URL 和 API Key
5. 点击 "添加"
6. 返回 "健康检查" 验证
7. 显示 ✅ 后可以使用
8. 在模型标签页中添加具体模型
```
