## 🏗️ 技术架构

### 日志流程

```
生成操作
    ↓
try-catch 包装
    ↓
成功 → addRenderLog(success)
失败 → addRenderLog(failed)
    ↓
setLogCallback 触发
    ↓
日志收集器接收
    ↓
[待 Phase 2] 日志面板显示
```

### 集成点

**StageAssets/index.tsx**
```typescript
handleGenerateAsset()      → 角色、场景生成
handleGenerateVariation()  → 服装变体生成
```

**StageDirector/index.tsx**
```typescript
handleGenerateKeyframe()   → 关键帧生成
handleGenerateVideo()      → 视频生成
```

**测试工具**
```typescript
test-logging.ts            → 日志收集和分析
```
