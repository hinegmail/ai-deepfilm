# AI 漫剧工场 (Print Film / AI Manga Studio)

## 产品概述

**AI 漫剧工场**是一个AI驱动的创意工作室，帮助创作者将故事创意快速转化为视觉资产和视频片段。面向短剧、漫剧、动画漫画和影视分镜的创作。

**核心价值主张：**

- 剧本→资产→关键帧工作流：先完成剧情 → 生成视觉资产 → 制作关键帧和视频
- 统一的AI工作台支持全流程生产
- 从故事创意快速迭代到可预览、可导出、可编辑的成果

## 目标用户

- 短剧创作者
- 漫画/漫画艺术家
- 分镜设计师
- 动画漫画制作者
- 影视视频概念创意团队

## 工作流程四大阶段

### Phase 01: 剧情创作

- 输入故事创意 → AI自动生成剧本结构、角色、场景、分镜
- 自动将文字描述转换为图像生成提示词
- 支持手动编辑剧情、角色、场景和视觉描述

### Phase 02: 场景角色

- 生成角色定妆设计表和服装变体
- 创建场景概念艺术和环境参考图
- 支持按角色或场景批量生成资产

### Phase 03: AI工作台

- 关键帧驱动工作流：为镜头生成起始帧和结束帧
- 基于关键帧进行视频生成，融合运动上下文
- 支持相机运动和复杂的舞台设置

### Phase 04: 制片导出

- 生成视频片段的时间轴可视化
- 预览和批量下载关键帧和视频
- 导出项目资产供后期剪辑使用

### 资产管理

- 所有提示词资产的集中查看和编辑
- 按角色、场景、镜头和状态搜索筛选

## 核心技术理念：关键帧驱动生成

传统的文字转视频模式难以稳定控制角色一致性和构图。本产品引入动画和影视制作原理：

1. **先生成关键帧** - 为每个镜头生成起始帧和结束帧
2. **插值生成运动** - 视频模型在关键帧之间生成运动过渡
3. **使用资产约束** - 参考角色定妆、服装变体和场景概念
4. **流程化生产** - 所有阶段共享同一项目状态（减少提示词复制）

## AI能力集成

- **文本模型** - 剧本拆解、角色/场景分析、提示词优化
- **图像模型** - 角色定妆、服装变体、场景、关键帧生成
- **视频模型** - 关键帧间的运动生成（支持Sora 2、Veo等模型）

模型可通过UI配置，支持不同的API提供商和模型版本。

## 数据架构

- **存储** - 浏览器IndexedDB存储所有项目数据（角色、场景、镜头、日志）
- **API密钥** - 本地存储，应用通过GitCC API代理调用
- **资产管理** - 支持Base64、远程URL和内联图像
- **无后端依赖** - 本地开发和桌面端通过HTTP代理实现
